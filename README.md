A simple Java console application that fetches and displays current weather information for a given city using a public weather API.

## Features
- Console-based input
- Fetches real-time weather data
- Displays temperature and region
- Uses environment variable for API key security

## Technologies Used
- Java 21
- Java HttpClient
- WeatherAPI.com

## How to Run
1. Set environment variable `WEATHER_API_KEY`
2. Run `WeatherApp`
3. Enter city name in console

## Notes
This project is created for learning purposes.
